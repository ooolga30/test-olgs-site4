Цифровые инструменты, которые стоит освоить учителю

Какие цифровые инструменты стоит освоить учителю

Платформа для просмотра и публикации видео YouTube

Что это такое, думаю, вы знаете. Задача для вас заключается в следующем:

  • найдите интересных людей на YouTube и подпишитесь на их каналы;
  • если видео нравится, надо его лайкать (увы, но педагогическая аудитория
    крайне скупа на проявление реакции в сетевом пространстве);
  • если возникают вопросы, надо их задавать в комментариях
    (это часть сетевой культуры);
  • оформить свой канал (не обязательно загружать видео,
    но поставить красивую заставку и аватар стоит).

Кладовая для вдохновения Pinterest

Это сервис для сохранения закладок — настоящая сокровищница для учителя.
  • создайте аккаунт;
  • найдите интересных пользователей и подпишитесь на них;
  • создайте свои коллекции.

Сервисы для создания учебных материалов для самопроверки

  • LearningApps
    • Технически он осваивается достаточно легко (это по обучалкам можно сделать
    за несколько минут). Его важно научиться мастерски использовать
    с педагогической точки зрения.
  • Quizlet
    • Идеальный сервис для запоминания иностранных слов или определения
    понятий/формул в игровой форме.
  • H5P
    • Функционал напоминает LearningApps, но возможностей гораздо больше.
